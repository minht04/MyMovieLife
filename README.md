# My Fashion Style

## サイト概要
自分のおすすめのファッションアイテムを投稿し、いいねやコメントができるサイト

### サイトテーマ
好きなファッションアイテムをを共有

### テーマを選んだ理由
自分自身服に興味があり、このファッションアイテムが好き！という気持ちを気軽に共有できるサイトがあればいいなと思ったため。

### ターゲットユーザ
服が好きな人

### 主な利用シーン
