## サイト概要
観た映画の感想を投稿するサイトです。<br>
人の感想に共感したものにはいいねやコメントをすることができます。<br>
気になったユーザーをフォローすることができます。


### サイトテーマ
映画の感想の投稿サイト

### テーマを選んだ理由
家で映画を観ることが増え、気軽に映画の感想を伝えるサイトがあれば良いなと思ったからです。<br>
また、利用者が自分の感想を人と共有することで、新たな価値観や気づきに出会えるのではないかと考えました。

### ターゲットユーザ
映画が好きな人

### 主な利用シーン
- 映画を観たあと、誰かに感想を伝えたいとき
- 観た映画の感想を記録したいとき
