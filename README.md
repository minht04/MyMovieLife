# My Fashion Style

## サイト概要
雑誌等から好きなファッションスタイルの画像を投稿し、いいねやコメントをすることができるサイト

### サイトテーマ
好きなファッションを共有

### テーマを選んだ理由
自分自身、服に興味があり、自分のコーディネートを投稿するサイトはあるがハードルが高く、気軽に共有できるサイトがあればいいなと思ったため。

### ターゲットユーザ
服が好きな人

### 主な利用シーン
いつでも、好きなときに
